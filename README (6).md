# Login and Registration System

## Overview
This project is a simple **Login and Registration System** built in C++. It allows users to register an account and log in securely using stored credentials. The program demonstrates basic file handling and authentication techniques.

## Features
- User registration with username and password
- Secure login verification
- File-based storage for user data
- Simple text-based user interface

## Requirements
- C++ compiler (e.g., GCC, MinGW, or Visual Studio)
- Basic knowledge of C++ programming and file handling

## How to Compile and Run
1. Open your terminal or command prompt.
2. Navigate to the folder containing the source file.
3. Compile the file using the following command:
   ```bash
   g++ "Login and Registration System.cpp" -o LoginSystem
   ```
4. Run the compiled executable:
   ```bash
   ./LoginSystem
   ```

## File Structure
- `Login and Registration System.cpp`: Main source code file for the Login and Registration System.
